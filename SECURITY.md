# Security Policy

## Supported Versions

Currently, we only provide security updates for the latest stable release of Essays.

| Version | Supported          |
| ------- | ------------------ |
| 2.8.x   | :white_check_mark: |
| < 2.8.0 | :x:                |

## Reporting a Vulnerability

We take the security of Essays seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, follow these steps:

1. **Email us**: Send a detailed report to [xuanlian@mac.com](mailto:xuanlian@mac.com).
2. **Details**: Include a description of the vulnerability, potential impact, and steps to reproduce.
3. **Response**: We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.

## Privacy Note

Essays is designed with privacy in mind:
- **On-device AI**: AI features use Apple Intelligence on-device processing. No data is sent to the cloud for these features.
- **Memos Data**: Your data is stored on your own Memos server. Essays only communicates with the server URL you provide.
